3. 데이터 타입 확인법
   
   데이터 타입은 `type(문자열)`로 확인할 수 있다.

   변수의 데이터 타입은 `print(type(변수))`로 확인할 수 있다. 
   결과값은 <class`데이터타입`>으로 나타난다.
